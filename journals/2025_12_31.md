- 今天注意到在 VS Code 中尝试用 `Cmd+Escape` 打开 Claude Code 扩展没反应，查 log 发现按下组合键时， `Escape` 键根本没送到 VS Code。
	- 在 System Settings → Keyboard → Keyboard Shortcuts → Mission Control 中翻到了 Game Overlay，它的默认快捷键是 `Cmd+Escape`，取消勾选就可以了。
	- [Game Overlay 是 macOS 26 Tahoe 引入的新功能](https://support.apple.com/en-us/105118#:~:text=full%20screen%2C%20press-,Command%2DEsc,-(Escape)%20to%20open)，快捷键冲突早在几个月前就[有人报告过了](https://discussions.apple.com/thread/256144225?sortBy=rank)，但我更新 macOS 26 也有一阵子了，直到今天才遇到，不清楚是最近某个更新默认启用了还是其他原因。
