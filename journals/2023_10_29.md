	- 在 Ruby 中**字符串与数字的比较会失败**
		- ```
		  5     <= 10     # => true
		  'abc' == 'def'  # => false
		  true            # => true
		  123456          # => true
		  0               # => true
		  nil             # => false
		  'xyz'.empty?    # => false
		  'a' > 5         # => error:
		                  # 字符串与数字的比较会失败
		  ```
	- 只有 `false` 和 `nil` 被视为“假”（falsey）
		- 所有其他的值都被视为“真”（truthy），这**包括数字 `0`**
