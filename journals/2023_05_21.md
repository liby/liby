- [JavaScript 中数组的 `.sort()` 方法的默认行为可能会导致一些问题，比如在处理数字数组时，它会默认将元素转换为字符串进行排序](https://twitter.com/ZoltanKochan/status/1660002475032686593)
	- 通过使用 TypedArray，可以避免默认的字符串排序行为
	- 当然对于数字排序，也可以向 `sort()` 方法传递一个比较函数以改变升序/降序顺序：
		- ```
		  [1,2,10].sort((a, b) => a - b);
		  [1,2,10].sort((a, b) => b - a);
		  ```
