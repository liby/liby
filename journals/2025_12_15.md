- VSCode 中设置 `"typescript.preferGoToSourceDefinition": true` 可以让 Cmd+Click 直接跳转到 npm 包的 JS 源码实现，而不是 .d.ts 类型定义文件（需要 TypeScript 4.7+）
	- ```settings.json
	  {
	    "typescript.preferGoToSourceDefinition": true,
	    "javascript.preferGoToSourceDefinition": true
	  }
	  ```
