	- 09-mutation
		- 解决 object mutation 带来的潜在问题
			- 方案 1：避免更改共享数据
				- ```js
				  let sherlock = {
				    surname: 'Holmes',
				    address: { city: 'London' }
				  };
				  
				  let john = {
				    surname: 'Watson',
				    address: sherlock.address
				  };
				  
				  john.surname = 'Lennon';
				  // 改变 john.address 指向的对象 city 的属性
				  john.address = { city: 'Malibu' };
				  ```
			- 方案 2：重新为变量分配新的对象
				- ```js
				  let sherlock = {
				    surname: 'Holmes',
				    address: { city: 'London' }
				  };
				  
				  let john = {
				    surname: 'Watson',
				    address: sherlock.address
				  };
				  
				  john = {
				    surname: 'Lennon',
				    address: { city: 'Malibu' }
				  };
				  ```
		- `const` 并不能阻止 object mutation
